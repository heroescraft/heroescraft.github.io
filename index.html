Here's my little troubleshoot:
(Note: Most devices can't be imported to, unless you jailbreak them.)

Pocket Edition, last I checked, couldn't be imported to, even though
it says it can, so try to import it on Pocket if you want, I don't think it
will work though. If importing on Pocket doesn't work you'll have to
change the file extension: ".mcpack" to ".zip"
Then you can unzip and move the world folder to:
On My Phone/Minecraft/games/com.mojang/MinecraftWorlds/


With Xbox I have no clue with. I've heard there's a web
browser and file system. If so then download the file on the Xbox
and navigate to your "Downloads" folder in your file browser. Try
opening that file (it should have a Minecraft like icon) and see if it
starts importing to Minecraft. If it can't then change the:
".mcpack" to ".zip" and extract.

Then (if you can find your Minecraft folder) and go to:
Minecraft/games/com.mojang/MinecraftWorlds/
Paste your minecraft world folder there.
(Sorry if your trying for Xbox, I'm not familiar with it.)

Windows 10 PC (or laptop) is definitely the easiest
out of all of them. All you have to do is download the
file, go to your downloads folder, open the file, and wait
for the export process to be done.


Just a note for all of these devices the Heroes of the Craft world is:
~800MB (~8/10GB)
So if you don't have that much space then good luck downloading!!
